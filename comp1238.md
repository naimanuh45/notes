## Week 1 – History of Computing
- Early computers were very large and expensive
- Computers evolved from mechanical to electronic systems

## Week 2 – Bits and Bytes
- Data is represented using binary (0s and 1s)
- A byte consists of 8 bits

## Week 3 – Command Line Basics
- The terminal allows direct interaction with the operating system
- Commands like ls, cd, and cat are used to navigate files

## Week 4 – Git and GitHub
- Git tracks changes in code over time
- GitHub allows collaboration and hosting repositories

## Week 5 – Web & Markdown
- Markdown is used to format text on GitHub
- README files act as documentation for repositories
